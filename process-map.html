v2 핵심 변경 (학생)

수준 진단이 시작점 — 기업 선택이 아닌 자기 진단부터 시작
달성 가능성 = 시간 x 노력 — 남은 학기를 고려한 동적 분석
복수 목표 설정 — 안정 1개 + 도전 1-2개, 최대 3개
통합 로드맵 — 공통 항목은 한 번만, 기업별 추가 항목 자동 병합

학년별 맞춤 분기

온보딩에서 학년 선택 → 탐색 모드(1~2) / 실전 모드(3~4) 자동 분기
1학년: 목표 기업 설정 + 기초 스펙 쌓기 (학점, 기초 어학)
2학년: 대외활동 · 공모전 참여 (경험 기록 집중)
3학년: 인턴 · 자격증 심화 (달성 분석 · 기업 매칭 시작)
4학년: 서류 · 면접 집중 (공채 일정 연동 · 단기 로드맵)

v2 핵심 변경 (관리자)
