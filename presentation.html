Problem

해결하려는 문제

학생이 센터를 찾지 않는다

"취업이 급하지 않으면 올 이유가 없다"는 인식. 저학년은 사실상 방치 상태.

지역 기업 정보가 없다

창원에 좋은 기업이 많지만 학생은 인재상·요구 스펙을 구체적으로 알 수 없음.

뭘 해야 할지 모르겠다

학점, 어학, 자격증, 인턴 — 다 해야 한다는데 순서도 모르고 계획도 없음.

내 수준을 모른 채 준비

목표 기업에 내가 얼마나 부족한지 모름. 막연히 스펙만 쌓다가 시간 낭비.

Talent ON2 / 12
Solution

Talent ON이 해결합니다

AI가 학생의 수준을 진단하고, 달성 가능한 기업을 매칭하며, 학기별 맞춤 로드맵을 자동 생성합니다.

수준 진단 → AI 기업 매칭

스펙 입력 후 남은 학기와 노력을 고려한 달성 가능성 기반 기업 추천

가중치 GAP 분석

필수/우대 스펙을 분리하고 가중치를 부여한 정교한 달성 분석

학년별 맞춤 로드맵

1~2학년 탐색 모드, 3~4학년 실전 모드로 분기하는 학기별 체크리스트

Grade Branching

학년별 맞춤 경험

탐색 모드 · 1~2학년

"뭘 준비해야 할지" 모르는 학생에게 방향 제시

  • 관심 분야 탐색 & 설정
  • 기업 둘러보기 (매칭이 아닌 탐색)
  • 기초 역량 체크리스트 (학점, 기초 어학)
  • 경험 기록 시작 (활동, 공모전)

실전 모드 · 3~4학년

"지금 내 수준으로 어디 갈 수 있지?" 에 답

  • AI 달성 가능성 기반 기업 매칭
  • 가중치 GAP 분석 (필수/우대 분리)
  • 단기 집중 로드맵 (자격증 시험일 연동)
  • 서류·면접 준비 (공채 일정 관리)

AI Features

3가지 AI 기능

AI-01

인재상 & 스펙 추출

채용공고 텍스트 → 구조화된 JSON
(필수/우대 스펙, 인재상 요약)

INPUT: 채용공고 원문
OUTPUT: { must, preferred, talent_profile }
AI-02

가중치 달성 분석

학생 스펙 vs 기업 요구 → 가중치 기반 GAP 리포트
+ 달성 가능성 점수

must_gap: -35점
preferred_gap: -15점
overall: 72/100
AI-03

맞춤 로드맵 생성

GAP 결과 + 학년 파라미터 → 학기별 체크리스트 자동 생성

grade_mode: junior | senior
steps: [{key, grade, semester,
category, title, priority}]
